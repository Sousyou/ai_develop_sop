# CURRENT

This file is the starter seed `CURRENT.md` for copied projects.
In this repository it is authored under `product/control/CURRENT.md` and is installed into copied projects as `.dev_sop/control/CURRENT.md`.

## Current phase
Bootstrap the current project onto the starter without mixing starter-owned core assets and project outputs.

## Current target
Use `core/*` as the reusable baseline and use the project surfaces for project-local outputs.

## Active slice
Establish or maintain the current project's working `plan -> spec -> implementation -> validation` loop.

## In scope now

- baseline rules under `core/rules/*`
- starter-owned templates, guides, examples, and skills under `core/*`
- project-local rules under `project-rules/*`
- project-local specs under `project-specs/*`
- project-local facts under `project-facts/*`
- recovery routing under `control/*`

## Out of scope now

- putting shipped-system or product docs under the installed SOP namespace
- turning `control/*` into a project board or work log
- writing project-local runtime outputs back into starter-owned `core/*`

## Current source of truth

- `control/DOC_MAP.md`
- `core/rules/rule-index.md`
- `project-rules/rule-index.md`
- `README.md`
- `VERSION.md`
- the active task spec in `project-specs/*.md`, if one exists
